# ミニかわロボ
手のひらサイズの格闘ロボット競技『ミニかわロボ』の公式リソース集です。

# 基本ルール抜粋
+ サイズ：100mm × 150mm以内（スタート時）。
+ 重量：150g以下。
+ 動力：マイクロサーボ（幅12mm）4個まで。
+ 勝敗：相手をひっくり返して10カウントか、場外に押し出せば勝利。

# 製作ガイド（初心者向け）
+ 推奨パーツリスト: FS90系サーボ、M5Atom S3、リチウムポリマー電池など、具体的で入手しやすい部品をリスト化。
+ 開発環境の構築: Arduino IDEの導入から、M5UnifiedやPS4Controllerライブラリのインストール手順を解説。
+ 「まずはこれを作ろう」: 3Dプリントデータとソースコードをセットにした、標準機体のサンプルを提示。

# 技術リソース（エンジニア向け）
+ 制御システム: PS4コントローラを使用したBluetooth操作や、IMUを用いた自動転倒検知のロジックを紹介。
+ 機構設計の工夫: やまば歯車（ダブルヘリカルギア）の活用や、ネジを極力使わないスナップフィット構造などの高度なテクニック。
+ M5Burnerの活用: プログラムを書き込むだけで動作するファームウェアの配布。

# イベント・ハンズオン主催者の方へ
+ 資料: 子供向けの操縦体験から、大人向けの0からの設計製作まで、対象に応じた実施案（を今後作成したいです。。）
+ 運営のヒント: 持ち運びやすいA3サイズのリングの作り方や、過去の大会（大阪、東京、滋賀など）の運営報告へのリンク。
